▶Console for time taken:-

1.The console.time() method starts a timer in the console view.
2.This method allows you to time certain operations in your code for testing purposes.
3.Use the console.timeEnd() method to end the timer and display the result in the console.view.
4.Use the label parameter to name the timer, then you are able to have many timers on the same page

▶Console for representing various data in table format:-

1.The console.table() method writes a table in the console view.
2.The first parameter is required, and must be either an object, or an array, containing data to fill the table.
3.The console.table() method represent in well mannered tabular format.
